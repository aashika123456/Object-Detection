# CodeAlpha Object Detection

## Project Overview

This project performs object detection using the YOLOv8 model. It can identify and locate multiple objects in an image by drawing bounding boxes and labels around detected objects.

## Features

* Object Detection using YOLOv8
* Detects multiple objects in an image
* Bounding Box Visualization
* Pre-trained Deep Learning Model
* Google Colab Implementation

## Technologies Used

* Python
* YOLOv8
* Ultralytics
* OpenCV

## How to Run

1. Install dependencies:
   pip install ultralytics

2. Upload an image.

3. Run the detection model.

4. View the detected output image.
